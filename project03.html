Arduino Weighing Scale

TThis project involves the development of a digital weighing scale using an Arduino microcontroller and a load cell sensor. The system is capable of accurately measuring weight and displaying the results on an LCD screen.

Technologies: Arduino, Load Cell, HX711 Module, LCD Display

Features:

  • Precise weight measurement using a load cell sensor
  • Real-time weight display on an LCD screen
  • Calibration function for improved accuracy
  • Lightweight and portable design
